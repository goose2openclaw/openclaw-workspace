## 三、专家模式深度推理 (5步法)

```
┌────────────────────────────────────────────────────────────────┐
│                 🧠 专家模式深度推理 (5步法)                       │
├────────────────────────────────────────────────────────────────┤
│                                                                 │
│  第1步: 市场状态分析                                             │
│    ├── 宏观数据 (CPI,利率,ETF流入)                               │
│    ├── 链上数据 (交易所余额,鲸鱼行为)                           │
│    ├── 技术面 (RSI,MACD,布林带,趋势线)                         │
│    └── 情绪面 (恐惧贪婪,资金费率,多空比)                        │
│                                                                 │
│  第2步: 声纳模型匹配                                            │
│    ├── 趋势识别 (上涨/下跌/震荡/突破)                           │
│    ├── 形态匹配 (W底,头肩顶,三角形整理)                         │
│    ├── 波动率分析 (高/中/低)                                    │
│    └── 相关性分析 (BTC带领/独立走势/轮动)                       │
│                                                                 │
│  第3步: 策略组合选择                                            │
│    ├── 保守策略 (只做BTC/ETH)                                   │
│    ├── 平衡策略 (主流+1-2个Alt)                                 │
│    ├── 激进策略 (全仓Alt高波动)                                  │
│    └── 对冲策略 (多空配对)                                      │
│                                                                 │
│  第4步: 风险评估计算                                            │
│    ├── 仓位公式: min(10%, 100/波动率²)                        │
│    ├── 止损公式: 入场价 - 2σ                                    │
│    ├── 止盈公式: 入场价 + 3σ                                    │
│    └── 风险系数: 波动率 × 持仓比例 × 杠杆                       │
│                                                                 │
│  第5步: 置信度综合评分                                           │
│    ├── 技术面: 40% (RSI/MACD/趋势)                            │
│    ├── 资金面: 30% (ETF/链上/主力)                            │
│    ├── 情绪面: 20% (恐惧贪婪/资金费率)                         │
│    └── 消息面: 10% (新闻/公告/社交)                            │
│                                                                 │
│  输出: 置信度 0-10 分                                          │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```

## 四、七大工具详细规范

### 🐰 打兔子 (Top20趋势追踪)

| 项目 | 规范 |
|------|------|
| **目标** | 追踪市值前20的主流币种趋势 |
| **选币** | 流通市值前20, 日均成交>1亿, 上市>1年 |
| **信号** | 20日均线>50日均线 (多头), 放量突破20日高点 |
| **仓位** | 基础10% + 信心加成 + 波动减成, 最大20% |
| **止损** | -8% |
| **止盈** | +30% 或 跌破20日均线 |

### 🐹 打地鼠 (高波动套利)

| 项目 | 规范 |
|------|------|
| **目标** | 捕捉高波动币种的短期机会 |
| **选币** | 24h波动率>8%, 成交量放大>200%, RSI极端 |
| **信号** | RSI<25买入, RSI>75卖出, 布林带触底/顶 |
| **仓位** | 基础5% + 波动加成 + 极端加成, 最大15% |
| **止损** | -5% |
| **止盈** | +15% |

### 🔮 走着瞧 (预测市场)

| 项目 | 规范 |
|------|------|
| **目标** | 利用Polymarket预测市场进行对冲/投机 |
| **选币** | 概率20-80%, 流动性>$10k, 关联代币 |
| **信号** | 概率低估, 事件临近, 极端情绪 |
| **仓位** | 基础3% + 概率加成 + 时间加成, 最大10% |
| **止损** | -10% |
| **止盈** | 概率回归或事件结束 |

### 👑 跟大哥 (做市协作)

| 项目 | 规范 |
|------|------|
| **目标** | 跟随大型做市商/机构的操作 |
| **信号** | 大单识别>$10万, 订单簿深度失衡>3:1, 负费率 |
| **仓位** | 基础8% + 大单加成 + 深度加成, 最大20% |
| **止损** | -5% |
| **止盈** | +10% 或 大单平仓 |

### 🍀 搭便车 (跟单分成)

| 项目 | 规范 |
|------|------|
| **目标** | 自动跟单盈利交易员 |
| **选币** | 胜率>60%, 收益回撤比>2, 最近30天盈利 |
| **信号** | 新开仓, 大仓位, 高置信度 |
| **仓位** | min(跟单仓位×20%, 5%), 最大10% |
| **止损** | 跟随交易员 |
| **止盈** | 跟随交易员 |

### 💰 薅羊毛 (新币空投)

| 项目 | 规范 |
|------|------|
| **目标** | 捕获加密生态的空投机会 |
| **选币** | 知名VC投资, 有代币空投预期, 上线前30天 |
| **信号** | GitHub活跃, Twitter增长, 测试网上线 |
| **仓位** | 基础1% + 热度加成 + VC加成, 最大5% |
| **策略** | 早期交互 → 申领 → 解锁卖出(80%) |

### 👶 穷孩子 (众包任务)

| 项目 | 规范 |
|------|------|

### 🛡️ 风控系统 (8大规则)

| 规则 | 限制 |
|------|------|
| 仓位上限 | 单币≤20%, 总仓≤60% |
| 止损纪律 | 硬止损-10%, 软止损-5%预警 |
| 波动率过滤 | σ>10%禁开, σ>15%强制减仓50% |
| 流动性检查 | 24h成交<100万USDT禁 |
| 相关性限制 | 相关>0.8视为同类, 同类≤25% |
| 时间过滤 | 22:00-02:00 UTC仓位减半 |
| 熔断机制 | 日亏5%暂停1h, 周亏10%暂停1天 |
| 黑名单 | 30天跌幅>50%禁做多 |
