# 🪿 中转钱包快速配置指南

## 什么是中转钱包？

中转钱包是平台提供的中间钱包，用于：
- 隔离用户主钱包
- 降低高风险操作风险
- 反向穿透审计

---

## 快速配置步骤

### 1. 创建中转钱包

```bash
POST /api/wallet/transit/create
{
  "name": "众包专用",
  "usage": "crowd"
}
```

### 2. 充值到中转钱包

- 从主钱包转账到中转钱包地址
- 建议首次充值: 0.1-0.5 ETH

### 3. 使用中转钱包

```bash
POST /api/wallet/transfer
{
  "from_wallet": "TW001",
  "to_address": "0x目标地址",
  "amount": 0.05
}
```

### 4. 反向审计

```bash
GET /api/wallet/audit/<transfer_id>
```

## 安全设置

- 单笔限额: 0.1 ETH
- 日限额: 0.5 ETH
- 需要审核: 是
- 自动销毁: 否
