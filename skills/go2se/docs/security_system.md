## 二、权限分级

### 2.1 令牌类型

| 令牌类型 | 权限 | 有效期 | 用途 |
|----------|------|--------|------|
| **Owner (所有者)** | 全部权限 | 1年 | 大白鹅CEO专用 |
| Backup Operator | backup, restore | 30天 | 运维人员 |
| Read Only | read | 7天 | 观察员 |

### 2.2 权限列表

```
Owner权限:
  ✅ read     - 读取数据
  ✅ write    - 修改配置
  ✅ backup   - 创建备份
  ✅ restore  - 恢复备份
  ✅ emergency - 紧急转移

Backup权限:
  ✅ read     - 读取数据
  ❌ write    - 修改配置
  ✅ backup   - 创建备份
  ✅ restore  - 恢复备份
  ❌ emergency - 紧急转移
```

## 三、资产保护

### 3.1 紧急转移流程

```
触发条件 → 身份验证 → 转移执行 → 记录日志 → 通知
```

### 3.2 转移记录

| 字段 | 说明 |
|------|------|
| ID | 唯一标识 |
| Amount | 转移金额 |
| Target | 目标地址(加密) |
| Reason | 转移原因 |
| Time | 时间戳 |
| Status | 状态 |

---

## 四、自动备份

### 4.1 备份类型

| 类型 | 触发 | 说明 |
|------|------|------|
| auto | 定时 | 每24小时 |
| manual | 手动 | 手动触发 |
| emergency | 紧急 | 紧急情况 |

### 4.2 备份内容

- 信号数据
- 配置参数
- 策略库
- API状态
- 访问日志

### 4.3 备份验证

```
创建 → 加密 → 校验和 → 存储 → 验证
```

## 五、安全机制

### 5.1 加密

| 项目 | 算法 |
|------|------|
| 数据加密 | AES-256 |
| 令牌生成 | SHA-256 |
| 地址存储 | 哈希不可逆 |

### 5.2 审计日志

```
时间 | 事件类型 | 详情 | 操作人
```
