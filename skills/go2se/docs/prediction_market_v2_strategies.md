# 走着瞧 v2 - 增强版 (竞品策略整合)

## 一、整合的竞品策略

### 1.1 六大策略模块

| 策略 | 权重 | 原理 | 适用场景 |
|------|------|------|----------|
| 均值回归 | 20% | 概率回归50% | 极端概率(>70%或<30%) |
| 动量策略 | 15% | 趋势持续 | 概率持续上升/下降 |
| 套利策略 | 25% | 跨市场价差 | 高流动性市场 |
| 事件驱动 | 20% | 事件前后交易 | 重大事件临近 |
| 流动性挖掘 | 10% | 优先高流动性 | 大成交量市场 |
| 情绪分析 | 10% | 社交媒体情绪 | 关键词情绪判断 |

## 二、机器学习模型

### 2.1 特征工程

```
输入特征:
- probability (概率)
- volume (成交量)
- days_to_end (距离到期天数)
- sentiment_score (情绪分数)
- momentum (动量)
- liquidity (流动性)

权重:
- probability: 25%
- sentiment: 20%
- momentum: 15%
- volume: 15%
- days_to_end: 15%
- liquidity: 10%
```

### 2.2 预测输出

```
prediction = 综合评分
confidence = 预测置信度
features = 特征详情
```

## 四、综合评分

### 4.1 评分公式

```
综合评分 = 策略评分×40% + ML置信度×35% + 高级分析×25%

策略评分 = Σ(策略分数 × 权重)
ML置信度 = 机器学习预测置信度
高级分析 = 技术分析综合得分
```

### 4.2 决策阈值

| 模式 | 置信度阈值 | 最小概率 | 信号数 |
|------|------------|----------|--------|
| 保守 | 70% | 65% | 1 |
| 平衡 | 55% | 55% | 5 |
| 激进 | 45% | 50% | 5 |
