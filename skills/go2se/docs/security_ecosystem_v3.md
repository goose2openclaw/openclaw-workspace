| 等级 | 名称 | 权限 |
|------|------|------|
| 5 | Owner | 全部权限 |

## 三、加密机制

### 3.1 哈希算法

```
SHA-256: 单次哈希
SHA-512: 强哈希
Chain: 10轮链式哈希
```

### 3.2 加密流程

```
数据 → JSON序列化 → Base64编码 → 链式哈希 → 存储
```

### 4.1 防御层级

```
Level 1: 白名单

Level 4: 异常检测
```

## 五、恢复系统

### 5.1 检查点

```
创建 → 验证 → 存储 → 索引
```

### 5.2 恢复程序

| 程序 | 说明 |
|------|------|
| full_reset | 完全重置 |
| partial | 部分恢复 |
| data_only | 仅数据恢复 |
