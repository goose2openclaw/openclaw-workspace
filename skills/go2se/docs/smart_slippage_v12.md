## 四、止盈/止亏逻辑

### 4.1 条件公式

```
止盈触发 = (盈利 >= 滑点阈值) AND (盈利 >= 最低盈利) AND (非专家模式 OR 非可移除模型)

止亏触发 = (盈利 <= -损失阈值)

专家模式 + 可移除模型 = 无止盈
```

### 4.2 默认配置

| 参数 | 值 |
|------|------|
| 滑点阈值 | 0.3% |
| 最低盈利 | 2% |
| 止损阈值 | 0.5% |
