## 二、声纳库 (SonarLibrary)

### 2.1 趋势模型

| 模型 | 准确率 | 类型 | 适用工具 |
|------|--------|------|----------|
| 牛旗 | 72% | bullish | rabbit, mole |
| 熊旗 | 68% | bearish | rabbit, mole |
| 双顶 | 75% | bearish | rabbit |
| 双底 | 73% | bullish | rabbit, mole |
| 突破 | 70% | bullish | rabbit, mole, follow |
| 跌破 | 68% | bearish | rabbit |
| 概率飙升 | 75% | bullish | prediction |
| 情绪转变 | 70% | neutral | prediction, follow |
| 价差扩大 | 65% | volatility | market_make |
| 流动性激增 | 72% | bullish | airdrop |

### 2.2 模型匹配

```
信号输入 → 工具识别 → 相关模型筛选 → 置信度计算 → 模型评分
```

## 三、策略库 (StrategyDB)

### 3.1 策略分类

| 工具 | 策略 |
|------|------|
| 走着瞧 | 均值回归、动量、套利、事件驱动、流动性、情绪 |
| 跟大哥/搭便车 | 顶尖表现、稳定性、风险调整、动量、分散化 |
| 跟大哥(做市) | 价差匹配、库存平衡、流动性、波动捕获、手续费 |
| 薅羊毛 | 价值最大化、效率、难度收益比、网络覆盖、领取时机 |
| 穷孩子 | 最高报酬、效率、简单、可用性、快速 |

### 3.2 策略权重

```
策略综合分 = Σ(策略分数 × 权重)
```

## 四、完整工作流

### 4.1 信号处理流程

```
┌─────────────────────────────────────────────────────────────────────┐
│  1. 数据输入                                                │
│     ↓                                                       │
│  2. 声纳模型匹配 (SonarLibrary)                           │
│     - 选择相关模型                                         │
│     - 计算置信度                                           │
│     - 输出模型评分                                         │
│     ↓                                                       │
│  3. 策略匹配 (StrategyDB)                                 │
│     - 获取工具相关策略                                      │
│     - 计算策略评分                                         │
│     ↓                                                       │
│  4. 置信度融合                                            │
│     - 声纳60% + 策略40%                                   │
│     ↓                                                       │
│  5. 回测分析 (BacktestEngine)                             │
│     - 100次模拟                                            │
│     - 计算胜率/盈亏                                         │
│     ↓                                                       │
│  6. 风控检查 (RiskController)                            │
│     - 仓位检查                                             │
│     - 亏损熔断                                             │
│     ↓                                                       │
│  7. 决策                                                   │
│     - EXECUTE / MONITOR / WAIT                           │
│     ↓                                                       │
│  8. 执行 + 中转钱包                                        │
└─────────────────────────────────────────────────────────────────────┘
```

### 4.2 决策规则

| 条件 | 决策 |
|------|------|
| 风控通过 + 置信度>50% + 胜率>50% | EXECUTE |
| 风控通过 + 置信度>50% + 胜率<50% | MONITOR |
| 其他 | WAIT |
