## 四、决策流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          决策流程                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  输入数据                                                                  │
│  ├── 实时行情 (价格/成交量/订单簿)                                        │
│  ├── 历史数据 (K线/指标)                                                  │
│  ├── 链上数据 (TVL/资金流)                                               │
│  ├── 社交情绪 (Twitter/Reddit)                                           │
│  └── 预测市场 (Polymarket)                                               │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 步骤1: 数据预处理                                                   │   │
│  │ • 去噪音 (MA/EMA/Kalman)                                          │   │
│  │ • 特征提取 (趋势/动量/波动率)                                      │   │
│  │ • 多时间框架聚合                                                    │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                      │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 步骤2: 趋势分析 (声纳模型)                                         │   │
│  │ • 形态识别 (100+趋势模型)                                         │   │
│  │ • 信号生成 (多因子加权)                                            │   │
│  │ • 置信度计算                                                        │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                      │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 步骤3: 策略匹配                                                     │   │
│  │ • 匹配投资行为 (打兔子/打地鼠/走着瞧/跟大哥/搭便车)              │   │
│  │ • 选择策略组合                                                       │   │
│  │ • 计算仓位                                                          │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                      │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 步骤4: 风险评估                                                      │   │
│  │ • 止损/止盈检查                                                     │   │
│  │ • 仓位超限检查                                                       │   │
│  │ • 备用金检查                                                         │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                      │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 步骤5: 执行                                                          │   │
│  │ • 买入/卖出/持有                                                    │   │
│  │ • 更新投资组合                                                       │   │
│  │ • 记录日志                                                          │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 五、参数阈值

### 5.1 可调参数范围

```python
PARAMETER_RANGES = {
    # 扫描参数
    'scan_interval': {'min': 30, 'max': 300, 'default': 60},           # 秒
    'confidence_threshold': {'min': 0.3, 'max': 0.9, 'default': 0.6},
    
    # 仓位参数
    'max_position': {'min': 0.01, 'max': 0.30, 'default': 0.10},
    'reserved_funds': {'min': 0.05, 'max': 0.30, 'default': 0.10},
    
    # 风险参数
    'stop_loss': {'min': 0.01, 'max': 0.10, 'default': 0.03},
    'take_profit': {'min': 0.05, 'max': 0.30, 'default': 0.08},
    
    # 策略参数
    'min_trend_probability': {'min': 0.3, 'max': 0.8, 'default': 0.5},
    'trend_change_threshold': {'min': 1, 'max': 5, 'default': 2},
    
    # 观测参数
    'min_observation_hours': {'min': 1, 'max': 12, 'default': 4},
    'max_observation_hours': {'min': 12, 'max': 72, 'default': 24},
    
    # 高频参数
    'hft_position_size': {'min': 0.001, 'max': 0.02, 'default': 0.005},
    'hft_profit_target': {'min': 0.001, 'max': 0.02, 'default': 0.005},
    'hft_stop_loss': {'min': 0.001, 'max': 0.01, 'default': 0.003},
}
```

### 5.2 模式配置

| 参数 | 保守 | 平衡 | 激进 |
|------|------|------|------|
| 扫描间隔 | 120s | 60s | 30s |
| 置信度阈值 | 0.7 | 0.6 | 0.5 |
| 最大仓位 | 8% | 10% | 15% |
| 止损 | 2% | 3% | 5% |
| 止盈 | 5% | 8% | 15% |
