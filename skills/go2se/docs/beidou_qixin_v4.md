## 四、决策引擎

### 4.1 决策流程

```
信号 → 回测分析 → 期望值计算 → 行动决策
              ↓
        模拟100次
        ├─ 胜率 = 胜利次数/100
        ├─ 平均回报 = 总回报/100
        └─ 期望值 = 胜率 × 平均回报
              ↓
        决策规则
        ├─ 期望值>1% 且 胜率>50% → EXECUTE
        ├─ 期望值>0 → MONITOR
        └─ 否则 → WAIT
```

### 4.2 行动类型

| 行动 | 条件 | 说明 |
|------|------|------|
| EXECUTE | 期望值>1% 且 胜率>50% | 立即执行 |
| MONITOR | 期望值>0 | 观察等待 |
| WAIT | 期望值≤0 | 不执行 |
