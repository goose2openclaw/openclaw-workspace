## 四、止盈移除条件

### 4.1 条件公式

```
当 满足以下所有条件 时，移除止盈限制：

1. 盈利达到最低要求 (min_profit)
2. 持仓时间达到短期时长 (duration_hours)
3. 从峰值回撤达到阈值 (drawdown_threshold)

公式:
  remove_tp = (drawdown >= threshold) AND (duration >= required) AND (profit >= min)
```

### 4.2 示例

```
打地鼠 (波动surf):
  - 盈利: 2% → 10%
  - 回撤: (10-7)/10 = 30%
  - 时长: 5小时 ≥ 2小时 ✓
  - 结果: 保持止盈 (回撤30%=阈值)

兔子 (突破):
  - 盈利: 3% → 10%
  - 回撤: (10-8)/10 = 20%
  - 时长: 4小时 = 4小时
  - 结果: 保持止盈 (回撤20%<25%)

跟大哥 (做市):
  - 盈利: 1% → 3%
  - 止盈: 无限制
  - 结果: ✅ 移除止盈
```

## 五、学习系统

### 5.1 自动学习

- 盈利交易 → 放宽止盈阈值
- 亏损交易 → 收紧止盈阈值
- 自动调整范围: 10%-50%

### 5.2 参数优化

```
调整 = ±2-3%
范围 = max(10%, min(50%, 当前值 + 调整))
```
