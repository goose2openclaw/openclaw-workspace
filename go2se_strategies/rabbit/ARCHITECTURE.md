## 参数配置

| 参数 | 值 | 说明 |
|------|------|------|
| MA短期 | 20日 | 短期均线 |
| MA长期 | 50日 | 长期均线 |
| 动量阈值 | 10% | 20日涨幅 |
| 波动率上限 | 5% | σ值 |
| 基础仓位 | 10% | - |
| 最大仓位 | 20% | - |
| 止损 | -8% | 硬止损 |
| 止盈 | +30% | 或跌破MA20 |
