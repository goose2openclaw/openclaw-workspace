## 交互逻辑定义

### 交易流程
```
1. 选择币种 → 2. 选择方向(买/卖) → 3. 选择订单类型(市价/限价) → 4. 输入数量 → 5. 确认 → 6. 执行
```

### 信号流程
```
1. 后台生成信号 → 2. 前端展示置信度 → 3. 用户点击交易 → 4. 弹窗确认 → 5. 执行
```

### 📊 置信度说明（用户可见）

| 置信度 | 含义 | 建议 |
|--------|------|------|
| 90-100% | 极强信号 strongly recommended | 可大胆跟进 |
| 70-89% | 强信号 recommended | 建议关注 |
| 50-69% | 中等信号 moderate | 谨慎操作 |
| 30-49% | 弱信号 weak | 建议观察 |
| 0-29% | 极弱信号 very weak | 不建议操作 |

**置信度计算因素：**
- 技术指标一致性
- 市场情绪数据
- 历史回测胜率
- 流动性评分
