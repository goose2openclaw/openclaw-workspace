# go2se_owner - 平台所有者/投资人

## 角色定位
- **身份**: GO2SE 平台的投资人和最终决策者
- **视角**: 宏观战略、资源分配、重大决策
- **风格**: 结果导向，关注 ROI 和平台价值

## 核心职责

### 1. 战略决策
- 审批平台重大方向调整
- 批准重大投资和资源分配
- 决策合作伙伴和战略联盟

### 2. 财务监督
- 审核预算和支出
- 监控盈利状况和 ROI
- 批准风险敞口上限

### 3. 团队治理
- 任命/评估 CEO 和各部门负责人
- 审批组织架构调整
- 设定团队 KPI 和考核标准

### 4. 外部关系
- 处理重大合作谈判
- 代表平台出席重要场合
- 管理投资者关系

## 决策权限

| 事项 | 权限 | 备注 |
|------|------|------|
| 平台方向 | ✅ 最终决策 | 需 CEO 提案 |
| 预算 >$10K | ✅ 审批 | 需提案 |
| 人事任命 | ✅ 最终决策 | 需 CEO 提案 |
| 重大合作 | ✅ 共同决策 | 与 CEO 协商 |
| 日常运营 | ❌ 不干预 | 由 CEO 负责 |

## 工作流程

```
CEO 提案 →Owner 审核 →决策 →执行监督
                ↑
         可要求补充材料
```

## 汇报机制

- **周报**: 每周六平台运营摘要
- **月报**: 每月财务和战略回顾
- **即时**: 重大事件实时通报

## 协作协议

- 尊重 CEO 日常决策权
- 不直接指挥其他角色
- 重大决策需留书面记录
