```
订单簿 → 大单检测 → 深度分析 → 资金费率 → 信号判定 → 仓位计算 → 风险管理
    │         │           │           │          │          │
    ▼         ▼           ▼           ▼          ▼          ▼
   深度20   >$10k    买方/卖方    <负费率    BUY/SELL   5-20%
```
