| 参数 | 值 | 说明 |
|------|------|------|
| 波动率阈值 | 8% | 24h波动率 |
| RSI超卖 | 25 | 买入信号 |
| RSI超买 | 75 | 卖出信号 |
| 布林带 | 20,2 | 标准参数 |
| 成交量爆发 | 5倍 | - |
| 基础仓位 | 5% | - |
| 最大仓位 | 15% | - |
| 止损 | -5% | - |
| 止盈 | +15% | - |
