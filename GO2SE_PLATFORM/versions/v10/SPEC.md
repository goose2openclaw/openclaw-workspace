## 设计理念 (继承v9)

### 核心理念
- **惊喜**: 大白鹅动画导入、信号视觉反馈、盈利庆祝效果
- **踏实**: 实时数据刷新、稳定面板更新、明确操作反馈
- **安全感**: 风控状态实时显示、资金变动确认、异常告警可视化

### 设计原则
- 字体: Sora / DM Sans / JetBrains Mono
- 色彩: OKLCH色彩空间，三模式主题
- 动效: cubic-bezier(0.16, 1, 0.3, 1)
- 交互: Focus状态清晰、Skeleton优于spinner

## 8个Sidebar模块详情

### 1. 📌 注意力板块
- 热点新闻/事件汇总
- DIY置顶功能 (Pin/Unpin)
- 一键清除/排序
- 快捷跳转到相关模块

### 2. 💰 资产看板
```
C1: 财产总额 + 增长率
C2: 收益额 (投资/打工分类)
C3: 钱包架构 (主钱包 + 7子钱包)
C4: 资产分布
    - 投资部分 (各工具仓位)
    - 打工部分
    - 保证金 (杠杆/跟大哥/搭便车)
    - 暂存平台 (无风险/保本套利)
    - 内部调配 (AI自动调度)
    - 转入/转出
C5: 资产风险状态、入金/出金记录、交易记录
C6: 模拟器 (回测/模拟/全向仿真)
```

### 3. 🎯 北斗七鑫 (投资+打工合并)
```
投资工具 (5种):
  🐰 打兔子 25% - 前20主流加密货币
  🐹 打地鼠 20% - 高频套利
  🔮 走着瞧 15% - 预测市场
  👑 跟大哥 15% - 做市协作
  🍀 搭便车 10% - 跟单分成

打工工具 (2种):
  💰 薅羊毛 3% - 空投猎手
  👶 穷孩子 2% - 众包打工

仪表盘:
  - 整体风格 (普通/专家 保守/平衡/积极)
  - AI推荐比例 (可滑动调整)
  - 各工具策略组合 (可调权重)
```

### 4. ⚙️ 工善其事 (信号+趋势+策略+仿真)
```
G信号输入:
  G1: 交易所API (Binance/OKX/Bybit)
  G2: Oracle及市场情绪 (MiroFish/CoinGecko)
  G3: 周期性仿真信号状态
      - 平台级别
      - 工具级别
      - 策略级别
      - 市场宏观/微观级别
  G4: 合作方API加入
  G5: 信号与缓存

H策略为王:
  H1: 策略逻辑架构图
  H2: 三风格选择 (保守/平衡/积极)
  H3: 当前策略组合 + 胜率
  H4: 仿真策略组合
  H5: 平台策略克隆 (3Commas/HaasOnline/FreqTrade)
  H6: 全部策略介绍 (123个趋势模型)
  H7: 声纳库趋势比照

N工程模式:
  N1: AI debug (MiroFish仿真)
  N2: 终端 Termux
  N3: 数据指令通道
```

### 5. 🔧 可迭代定制化平台
```
J1: 竞品集成 (一键加入)
    - 3Commas / HaasOnline / FreqTrade
J2: 学习及监控竞品胜率
J3: 私募LP/专家定制
    - 交易习惯学习
    - 灵活配置风格
    - 策略倾向
    - 保密安全
    - 自定义专家模式

防衰因子:
    - 因子退化度监控
    - 自动优化触发
    - 版本回滚点
```

### 6. 🛡️ 安全机制
```
I1: 前端安全 (XSS/CSRF防护)
I2: 后门严格管控
I3: 左右脑防崩溃 (双版本备份)
I4: 资金安全 (多重签名/冷却期)
I5: 数据和算力安全
I6: 日常和优化前自动全向仿真
I7: 不同版本客户端适应性
```

### 7. ⚙️ 设置 (设置+私募LP+系统版本)
```
L1: 钱包架构设置/链接/状态
L2: 交易所、基金等机构 (CEX/DEX)
L3: 信息同步 (多种IM + 智能办公)
L4: OPC (一人公司) 托管

K私募LP:
  K1: 私募部落 (招募基金管理人群组)
  K2: 平台自身私募LP
      - 不同权限和资源
      - 对应分润机制

M系统版本:
  M1: 左右双脑备份 (可设不同版本)
  M2: 最优状态记录 (收益/胜率/抗压/记忆点)
  M3: 可逆时间点 (升级优化前)
  M4: 脚本日志输出
```

### 8. 📞 联系与转发
```
O1: 客服Bot
O2: 平台自动下载链接和二维码
O3: 制作OpenClaw和GO2SE量化盘
O4: 转发有礼 (邀请好友奖励)
```
