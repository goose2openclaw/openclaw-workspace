## 升级路径

```
游客 → 订阅 → 会员 → 鹅伙人 → 专家
         ↓
      专家模式 (独立路径)
```

### 升级条件

| 升级 | 条件 |
|------|------|
| 游客→订阅 | 注册+手机验证 |
| 订阅→会员 | 交易满$1000 |
| 会员→鹅伙人 | 入金>$50000 |
| 鹅伙人→专家 | 审核+面试 |
| →专家模式 | 付费或邀请 |
