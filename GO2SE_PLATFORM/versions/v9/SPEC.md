## 设计理念

### 核心理念
- **惊喜**: 首次访问大白鹅动画导入、信号视觉反馈、盈利庆祝效果
- **踏实**: 实时数据刷新、稳定面板更新、明确操作反馈
- **安全感**: 风控状态实时显示、资金变动确认、异常告警可视化

### 设计原则 (参考 frontend-design-pro)
- 字体: Geist / DM Sans / Sora (有个性，不用Inter/Arial)
- 色彩: OKLCH色彩空间，背景用 #0f0f0f (暗色模式)，中性色带色调
- 空间: 4px/8px基础间距，内容宽度限制65ch
- 动效: cubic-bezier(0.16, 1, 0.3, 1)，100-200ms微交互，300-500ms页面过渡
- 交互: Focus状态清晰、Skeleton优于spinner、错误信息具体可操作

## 启动动画规格

### 大白鹅导入动画
```
阶段1: Logo显示 (0-500ms)
  - 大白鹅SVG图标 + "GO2SE" 文字
  - 脉冲发光效果

阶段2: 灵魂定位 (500-1000ms)
  - 渐显 "AI优雅薅羊毛 / Crypto量化投资"
  - 淡入淡出效果

阶段3: 进度条 (1000-3000ms)
  - 文字: "红掌拨清波ing..."
  - 渐变进度条 (翡翠绿 #00D4AA)
  - 百分比数字跳动

阶段4: 页面淡入 (3000-3500ms)
  - 整体fade-in
  - 进度条消失
```

## v9.1 迭代计划 (根据gstack评审)

### 评审结论
- **整体评分**: 7.2/10
- **核心问题**: 18个一级入口过载（业界标准4-8个）
- **建议**: 砍到5-8个一级入口，其余按需解锁

### MVP优先模块 (P0)
1. a注意力板块 - 用户首屏，决定留存
2. c资产看板 - 核心价值展示
3. d投资配置 - 交易核心
4. i安全机制 - 资金安全底线
5. 注册/登录 - 不登录啥都干不了

### 重构方案
```
Sidebar重构: 15模块 → 4-6主模块 + 子菜单

首页 (Dashboard)
  └─ a注意力板块 + 资产总览快捷入口

投资 (Trading)
  └─ c资产看板 + d投资配置 + e打工配置 + h策略为王

工具 (Tools)
  └─ f算力和资源 + g信号输入 + i安全机制

我的 (Account)
  └─ k私募LP + l设置 + o联系转发

[隐藏/专家模式]
  └─ j机器学习 + m系统版本 + n工程模式
```
