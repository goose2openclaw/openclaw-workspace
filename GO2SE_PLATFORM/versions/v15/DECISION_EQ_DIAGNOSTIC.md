## 四、优化方案

### 🔧 方案A: 降低Mi上限 (推荐快速修复)

```python
# 新Mi公式: 限制放大倍数
Mi = 0.85 + 0.30 × (当前评分/100)
    = 0.85 + 0.30 × 0.934 = 1.130

# 中性(avg=0.00):   Final = 1.130 × 0.00 = 0.000 → HOLD ✅
```

### 🔧 方案B: 改用符号共识 (推荐长期方案)

```python
# 方向由Σ(wi×Si)符号决定, Mi只调整幅度
direction_sign = sign(Σ(wi × Si))  # +1, -1, 或 0
signal_strength = |Σ(wi × Si)| / Σ(wi)  # 0-1
consensus_score = Σ(wi × Si) / Σ(wi)  # 带符号

Final = consensus_score × Mi × Ri
      = direction_sign × signal_strength × Mi × Ri

# Mi只放大信号强度，不改变方向
```

### 🔧 方案C: 阈值自适应 (保守)

```python
# 高Mi时收紧阈值
THRESHOLD_LONG  = 0.70 + (Mi - 1.0) × 0.5   # Mi=1.43 → 0.915
THRESHOLD_SHORT = 0.30 - (Mi - 1.0) × 0.3   # Mi=1.43 → 0.171
```

## 六、已验证的决策等式

