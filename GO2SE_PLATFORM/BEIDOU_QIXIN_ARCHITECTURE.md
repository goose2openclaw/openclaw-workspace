## 二、七种工具详解

### 2.1 投资工具

#### 🐰 打兔子 (Top 20 主流加密货币)
```
性质: 前20主流加密货币，参考主流股票证券
仓位: 25% (可调范围 15-35%)
止损: 5% | 止盈: 8%
置信度: ≥70% 执行
策略: 趋势跟踪 + 突破策略
评分因子: 主流币波动率、成交量、RSI、均线排列
动态调整: 趋势恶化时自动减仓≤15%
参考基准: 股票证券的稳健收益策略
```

#### 🐹 打地鼠 (其他所有加密货币)
```
性质: 非前20加密货币，扫描发现异动
仓位: 20% (可调范围 10-30%)
止损: 8% | 止盈: 15%
核心: 火控雷达锁定异动标的
专家模式: 止盈点可条件移除 (达标条件预设)
扫描: 24/7全市场异动监控
评分因子: 异动强度、成交量突变、RSI极端值
```

#### 🔮 走着瞧 (预测市场)
```
性质: 预测市场 + MiroFish仿真指导
仓位: 15% (可调范围 10-25%)
止损: 5% | 止盈: 10%
核心: 寻找胜率>65%的预测内容
集成: MiroFish仿真 + 现有策略对比
数据源: Polymarket, Gemini, AI21
评分: 预测胜率 × 置信度 × 市场活跃度
```

#### 👑 跟大哥 (做市协作)
```
性质: 跟单做市协作
仓位: 15% (可调范围 10-20%)
止损: 3% | 止盈: 6%
核心: MiroFish评估 + 综合判断
策略: 跟随优质做市商信号
风控: 独立评估 + 最大跟单上限
```

#### 🍀 搭便车 (跟单分成)
```
性质: 跟单分成，二级分包
仓位: 10% (可调范围 5-15%)
止损: 5% | 止盈: 8%
核心: 加强风控 + 策略仿真
特点: 二级分包降低风险
评分: 信号质量 × 跟单人数 × 历史表现
```

### 2.2 打工工具

#### 💰 薅羊毛 (空投猎手)
```
性质: 空投猎手，大范围快速抓空投
仓位: 3% (可调范围 1-5%)
止损: 2% | 止盈: 20%
核心原则:
  ⚠️ 杜绝授权链接 (approve/signature请求一律拒绝)
  ⚠️ 只做只读API调用
  ✅ 中转钱包严格风控
  ✅ 背景和数据调研先行
  ✅ 风险评测后才行动
技能: airdrop_hunter.py (ML Core内置)
```

#### 👶 穷孩子 (众包赚钱)
```
性质: 众包赚钱机会
仓位: 2% (可调范围 1-3%)
止损: 1% | 止盈: 30%
核心:
  ✅ 利用EvoMap tools等社交圈
  ✅ 做好人设获得更高收益任务
  ✅ 隔离保护 (工作与个人分离)
  ✅ 众包任务筛选 + 风险评估
技能: crowdsignal.py (ML Core内置)
```

## 三、趋势判断层 (从顶向下)

### 3.1 声纳库 (Sonar)
```
位置: backend/app/core/sonar_v2.py
规模: 123个趋势模型 (持续扩充)
分类:
  - 趋势类 (EMA交叉、均线排列、通道)
  - 动量类 (RSI、MACD、ADX)
  - 波动类 (布林带、ATR)
  - 成交量类 (量价齐升/齐跌)
  - 反转类 (CCI、威廉姆斯%R)
扫描流程:
  1. 定焦扫描信号 → 比照模型组
  2. 扫描比照 → 缩小范围
  3. 高置信度判断趋势/趋势组合
  4. 自动触发对应工具的投资策略
```


## 十一、声纳库 123趋势模型

### 模型分类统计
```
趋势类 (EMA/MA/通道): ~30个
动量类 (RSI/MACD/ADX): ~25个
波动类 (Bollinger/ATR): ~20个
成交量类 (OBV/VWAP): ~15个
反转类 (CCI/Williams %R): ~15个
通道类 (Channel/Breakout): ~10个
其他专用类: ~8个
总计: 123个
```

### 核心扫描流程
```
1. Layer1粗筛 ( coarse_scan )
   → 快速扫描全部123模型
   → 筛选出潜在匹配模型

2. Layer2精筛 ( evaluate_category_signal )
   → 对粗筛结果精细评分
   → 计算置信度

3. 趋势组合判断
   → 多模型共识
   → 高置信度趋势输出

4. 策略触发
   → 趋势 → 工具映射
   → 自动触发对应策略
```
