## 🔄 从dry_run切换到live

### 步骤1: 确认配置
```bash
cat backend/.env
# 确保 TRADING_MODE=dry_run
# 确保 API密钥已配置
```

### 步骤2: 小额测试
```bash
# 先用小资金测试
MAX_POSITION=0.1  # 只用10%仓位
```

### 步骤3: 切换到live
```bash
# 编辑 backend/.env
TRADING_MODE=live

# 重启服务
./scripts/deploy_production.sh <API_KEY> <SECRET_KEY> live
```

### 步骤4: 监控
```bash
# 监控日志
tail -f backend.log

# 检查健康状态
curl localhost:8000/api/health
```
