# v6i vs v15 收益率差距根因分析
## +42.42% vs +13.68% — 为什么不一样

---

## 核心发现：交易频率是关键

| 指标 | v6i | v15 |
|------|------|-----|
| 30天收益率 | +42.42% | +13.68% |
| 交易次数 | **574笔** | **8笔** |
| 日均交易 | **19.1笔/天** | 0.27笔/天 |
| 平均每笔收益 | +0.85% | +1.71% |
| 胜率 | 69.3% | **100%** |
| 最大回撤 | 2.3% | 12.03% |
| 策略类型 | **高频刷单** | **低频精准** |

---

## 根因：v6i靠高频，v15靠精准

### v6i收益拆解
```
打地鼠(SOL): 370笔, +60.2%  (主力收益来源)
打兔子(BTC): 161笔, +23.9%
打兔子(ETH): 43笔,  +6.5%
空投打工:   +892 USDT/月
─────────────────────────────
合计: 574笔 → +42.42%
```

v6i的+42%不是靠单笔大赚，而是靠 **"积小胜为大胜"**：
- 每笔平均收益 +0.85%
- 574笔 × 0.85% = +487.9% 理论收益
- 扣除损耗后 = +42.42% 实盘

### v15收益拆解
```
做多(LONG): 4笔, 平均+2.1%/笔
做空(SHORT): 4笔, 平均+2.5%/笔
─────────────────────────────
合计: 8笔 → +13.68%
```

v15虽然**单笔质量更高**（均收益1.71% > v6i的0.85%），但**交易次数太少**（8笔 vs 574笔）

## 收益率公式拆解

```
总收益 = Σ(每笔收益)
       = 交易次数 × 平均每笔收益 × 胜率 × 平均盈亏比

v6i: 574笔 × 0.85% × 69.3% × 1.28 = +42.42%
v15:   8笔 × 1.71% × 100%  × 1.50 = +13.68%
```

**关键变量**:
- v6i靠 **交易次数** (574笔)
- v15靠 **单笔质量** (均收益1.71% > v6i的0.85%)

## 结论

| 对比项 | v6i | v15 |
|--------|-----|-----|
| 策略哲学 | 高频刷单 | 精准低频 |
| 交易次数 | 574笔 | 8笔 |
| 单笔质量 | 0.85% | **1.71%** ✅ |
| 胜率 | 69.3% | **100%** ✅ |
| 总收益 | **42.42%** | 13.68% |
| 最大回撤 | **2.3%** | 12.03% |

**v15不是不如v6i，而是走了不同的路**:
- v6i: **宽度**策略 — 广撒网，多工具，高频
- v15: **深度**策略 — 深研究，精准，低频

**要提升v15收益，最有效的方法是降低交易频率门槛，增加交易次数，同时保持高胜率**
