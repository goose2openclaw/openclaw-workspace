## 📋 概述

基于趋势 + MiroFish预测 + 矿工费 + 归一胜率的自动仓位优化系统。

### 优化因子

| 因子 | 公式 | 影响 |
|------|------|------|
| **矿工费** | Taker 0.1% × 2 (双向) | 降低净收益 |
| **归一胜率** | (WR - min) / (max - min) × 0.6 + 0.3 | 相对排名权重 |
| **趋势调整** | 信号 × 置信度 × 20% | 看涨增配/看跌减配 |
| **MiroFish** | (信号 - 0.5) × 置信度 × 30% | 信号强度权重 |

## 🧮 归一胜率计算

```python
归一胜率 = (胜率 - 最小胜率) / (最大胜率 - 最小胜率) × 0.6 + 0.3

示例:
- 打地鼠胜率: 55% → 归一化后: 83.8%
- 跟大哥胜率: 50% → 归一化后: 77.5%
- 打兔子胜率: 12% → 归一化后: 30.0% (最低)
```

## ⚙️ 动态调仓逻辑

### 趋势信号触发

```
趋势 = 看涨(+1) → 权重 × (1 + 置信度 × 0.5)
趋势 = 中性(0)  → 权重不变
趋势 = 看跌(-1) → 权重 × (1 - 置信度 × 0.7)
趋势 = 强看跌  → 权重 × 0.1 (几乎禁用)
```

### MiroFish信号触发

```
信号 > 0.75 (强买) → 权重 × 1.375
信号 = 0.50 (持有) → 权重 × 1.0
信号 < 0.25 (强卖) → 权重 × 0.625
```
