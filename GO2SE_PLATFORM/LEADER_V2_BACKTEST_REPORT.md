## 五、做空策略详解

### 1. 趋势跟踪做空
```yaml
适用场景: 下跌趋势确认后
入场条件:
  - 评分 < 70
  - 价差 > 0.5%
  - 交易量 > $1M
止损: 5%
止盈(普通): 6%
止盈(专家): 无限制
```

### 2. 均值回归做空
```yaml
适用场景: 价格偏离均值时
入场条件:
  - RSI > 70
  - 布林带突破上轨
  - 价差 > 0.4%
止损: 4%
止盈(普通): 8%
止盈(专家): 无限制
```

### 3. 流动性做空
```yaml
适用场景: 大额卖出时
入场条件:
  - Whale大额转账
  - 流动性骤降
  - 价差扩大
止损: 6%
止盈(普通): 10%
止盈(专家): 无限制
```

### 4. 价差做市
```yaml
适用场景: 双向提供流动性
策略:
  - 低价差时做多
  - 高价差时做空
止损: 3%
止盈(普通): 5%
止盈(专家): 无限制
```
