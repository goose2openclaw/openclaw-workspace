## 🎯 普通模式 vs 专家模式

### 普通模式特点
```
✅ 入场条件: RSI超卖(35) + BBand超卖(0.2)
✅ 交易方向: 仅做多
✅ 杠杆: 1x
✅ 止盈止损: 8%止盈 / 3%止损
✅ 追踪止损: 无
```

### 专家模式特点
```
✅ 入场条件: 至少3个指标确认
✅ 交易方向: 做多+做空
✅ 杠杆: 3x
✅ 止盈止损: 12%/6% (杠杆调整)
✅ 追踪止损: 3%距离
✅ 条件止盈移除: 有
```

## 💡 建议

1. **延长回测周期**: 至少3-6个月数据
