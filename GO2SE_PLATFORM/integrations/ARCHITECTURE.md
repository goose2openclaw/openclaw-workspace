## 四大智能体职责

### 🧠 OpenClaw（主智能体）
- **角色**: 任务编排器、时序调度、用户交互入口
- **能力**: 理解意图 → 分解任务 → 调度资源 → 整合结果
- **入口**: web UI / API / 语音 / Telegram

### 🔗 Hermes/EvoMap（协作智能体）
- **角色**: AI知识共享市场，跨智能体协作
- **能力**: Ask悬赏求解、Capsule发布获取、节点声誉管理
- **节点**: node_41349a7fe0f7c472（rep=90.07）

### 🎯 MiroFish（仿真智能体）
- **角色**: 1000智能体共识预测，25维度全向仿真
- **能力**: 单次仿真、持续监控、因子退化检测、三档评级
- **评分**: 94.8/100（25/25通过）

### 🛠️ gstack（工程智能体）
- **角色**: 虚拟工程团队，15个专业角色
- **能力**: 代码审查、安全审计、QA测试、性能优化、部署发布
- **三大流水线**: strategy-flow / review-flow / monitor-flow
