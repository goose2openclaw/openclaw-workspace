## 二、决策机制

### v6i 决策机制
```
信号强度: confidence = f(RSI)
  RSI < 30: confidence = (30-RSI)/30 * 10 + 5  → 强做多信号
  RSI > 70: confidence = (RSI-70)/30 * 10 + 5  → 强做空信号
  中性区间: confidence = 5 (中)

投票机制: 7个Agent各自投票 → 简单多数决
决策规则:
  Normal模式: confidence > 65% + regime=bull → LONG
  Expert模式: confidence > 65% → LONG/SHORT自动切换
```

### v15 决策机制
```
优化决策等式:
  Final = sign(Σ wi×Si) × |Σ wi×Si/Σ wi| × min(Mi, 1.20) × Ri

其中:
  wi  = 四脑权重 [alpha:0.25, beta:0.25, gamma:0.30, delta:0.20]
  Si  = 脑信号强度 (-1, -0.5, 0, +0.5, +1)
  Mi  = MiroFish 25维系数 (上限1.20)
  Ri  = 风险系数 (regime × RSI权重 / 波动率)

决策规则:
  |Final| < 0.20 → HOLD (死区过滤)
  Final > 0.75 → LONG
  Final < 0.25 → SHORT
```
