## 🐰 打兔子策略 - 主流币趋势跟踪

### 策略概述

| 属性 | 值 |
|------|-----|
| **名称** | 🐰 打兔子 (Rabbit) |
| **版本** | v2.0 |
| **性质** | 趋势跟踪 + 趋势反转 |
| **目标市场** | 前20主流加密货币 |
| **持仓周期** | 1-7天 |
| **风险等级** | 🟡 中低 |

### 目标币种 (20个)

```
BTC, ETH, BNB, XRP, SOL, ADA, DOGE, AVAX, DOT, MATIC,
LINK, UNI, ATOM, LTC, ETC, XLM, ALGO, VET, ICP, FIL
```

### 核心理念

> "像兔子一样敏捷，在主流币趋势中快速捕获收益"

- **专注主流**: 只做前20大币种，流动性好、波动可预测
- **趋势为王**: 顺势而为，不抄底、不摸顶
- **快进快出**: 持仓不超过7天，避免长时间暴露

### 风险参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 止损 | **5%** | 亏损达到5%强制平仓 |
| 止盈 | **8%** | 盈利达到8%锁定利润 |
| 最大单币仓位 | 10% | 单个币种不超过总资金10% |
| 最大总仓位 | 25% | 全部兔子仓位不超过25% |
| 最低置信度 | 65% | 信号低于此值不执行 |
| 最大持仓天数 | 7天 | 超时强制平仓 |

### 买入条件 (需同时满足)

```
✅ 条件1: MA7 > MA25 (均线多头排列)
✅ 条件2: 当前价 > MA7 (价格站稳短期均线上)

✅ 条件4: 成交量 > 平均成交量 × 1.5 (放量确认)

```
🛑 止损: 亏损 ≥ 5%

⏰ 时间止损: 持有 ≥ 7天

### 仓位计算公式

```python
def calculate_position(confidence: float, price: float, total_capital: float) -> float:
    """
    置信度越高，仓位越大
    最大仓位: 10% (confidence >= 0.9)
    标准仓位: 5% (0.7 <= confidence < 0.9)
    轻仓: 2% (0.65 <= confidence < 0.7)
    """
    if confidence >= 0.9:
        size = 0.10
    elif confidence >= 0.7:
        size = 0.05
    elif confidence >= 0.65:
        size = 0.02
    else:
        size = 0.0
    
    return min(size * total_capital, total_capital * 0.10)
```

### 仓位计算公式

```python
def calculate_position(volatility: float, rsi: float, total_capital: float) -> float:
    """
    波动率越大，仓位越小 (风险控制)
    RSI极端时，增加仓位 (均值回归概率高)
    
    公式:
    仓位 = 基础仓位(5%) + 波动加成 + 极端加成
    """
    # 基础仓位
    position = 0.05
    
    # 波动加成: 波动率 × 0.5%
    # 8%波动 → +4%
    # 15%波动 → +7.5%
    volatility_bonus = volatility * 0.5
    position += volatility_bonus
    
    # 极端加成: RSI<20 或 RSI>80
    if rsi < 20 or rsi > 80:
        position += 0.03
    
    # 限制在 [2%, 15%]
    return max(0.02, min(position, 0.15))
```
