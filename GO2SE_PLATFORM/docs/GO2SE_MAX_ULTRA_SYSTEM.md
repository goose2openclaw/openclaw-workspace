## 七、自主迭代循环

### 每日循环

```
1. 数据收集 (每15分钟)
   → 市场数据、情绪数据、策略表现

2. 权重更新 (每2小时)
   → 外部胜率抓取、MiroFish仿真

3. 策略回测 (每日收盘)
   → 历史数据回测、参数优化

4. 风险评估 (每小时)
```
